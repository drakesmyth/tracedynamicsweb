# Archive — original Mobirise site

Media and documents harvested from the **original TRACE marketing site**
(the Mobirise-built static site that lived on the `main` branch of the
`tracekinodynamics.github.io` repository) before that repo was retired.

Preserved here so this repo is the single home for the project's site assets.
These files are **not** used by the current Nuxt site (`../../trace-site/`) —
they are kept for reference and reuse.

## Contents

- **`media/`** — content and hardware photography, logos, screenshots, and the
  Mobirise theme background images from `assets/images/` of the old site.
- **`docs/`** — source documents:
  - `references.md`
  - `Future of labor.pages` (Apple Pages)
  - `faq0.1.pages` (Apple Pages)

## Notes

- The old site's `tracemedia/` folder is already present at this repo's root
  (`../../tracemedia/`), so it was not duplicated here.
- Filenames are preserved verbatim from the Mobirise export (hence the
  size-suffixed names like `cliff.png-1-495x498.png`).
- Original source of record: `tracekinodynamics.github.io` @ `main`
  (Mobirise Website Builder export).
