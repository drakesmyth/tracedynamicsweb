# TRACE Website

## Goal

Build a production-quality website for TRACE Dynamics that also serves as a flagship portfolio project.

The design language should resemble the Mistral AI website:
- clean
- minimal
- modern
- premium
- lots of whitespace
- subtle gradients
- rounded cards
- editorial typography

This is NOT a clone.

## Tech Stack

Framework:
- Nuxt 4
- Vue 3
- TypeScript

Styling:
- TailwindCSS
- GSAP
- Lenis
- Three.js / TresJS

Commerce
- Shopify Storefront API
- Stripe if needed
